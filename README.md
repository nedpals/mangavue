# mangavue
A simple manga reader powered by the whole Vuejs ecosystem (Vuex-Vue-router-Vuejs)
API: https://market.mashape.com/doodle/manga-scraper

To start:

```bash
$ yarn install
```

To develop:

```bash
$ yarn run dev```

To build for production:

```bash
$ yarn run build
```

To lint you code:

```bash
$ yarn run lint
```
